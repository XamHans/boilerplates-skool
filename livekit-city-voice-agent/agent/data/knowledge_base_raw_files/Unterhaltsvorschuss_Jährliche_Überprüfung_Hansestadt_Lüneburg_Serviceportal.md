# Unterhaltsvorschuss - Jährliche Überprüfung

## Allgemeine Informationen

Sie sind alleinerziehend und Ihr Kind erhält Unterhaltsvorschuss vom Jugendamt.  
Jedes Jahr muss einmal geprüft werden, ob Sie und Ihr Kind die Voraussetzungen dafür weiterhin erfüllen.  
Für diese jährliche Prüfung können Sie mit unserem [Online-Dienst](LINK) Ihre Angaben machen.
