# Boilerplates Skool

A monorepo collecting starter templates and boilerplates for the Skool community.

## Projects

| Folder | Description |
|--------|-------------|
| `fullstack-ai-starter` | Full-stack AI starter template |
| `livekit-city-voice-agent` | LiveKit voice agent boilerplate — imaginary city support use case with RAG |

## Usage

Each project is self-contained. Navigate into the folder and follow its own README.

```bash
cd fullstack-ai-starter
# or
cd livekit-city-voice-agent
```
