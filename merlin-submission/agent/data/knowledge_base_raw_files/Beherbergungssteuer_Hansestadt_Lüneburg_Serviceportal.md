# Beherbergungssteuer

## Abgabe von Steuererklärung zur Beherbergungssteuer

- [Abgabe von Steuererklärung zur Beherbergungssteuer (mit dem Personalausweis)](LINK) - Online-Service
- [Abgabe von Steuererklärung zur Beherbergungssteuer (mit ELSTER-Zertifikat)](LINK) - Online-Service

### Teaser

Für Beherbergungsbetriebe besteht eine Steuerpflicht.  
Die Steuererklärung können Sie ohne Papier und Besuch bei uns mit unseren Online-Diensten abgeben.  
Umfangreiche Informationen zur Nutzung der Online-Dienste haben wir Ihnen auf unserer [Informationsseite zu den Online-Diensten Beherbergungssteuer bereitgestellt.](LINK)

### Allgemeine Informationen

Eine Steuererklärung ist bis zum 15. Kalendertag nach Ablauf eines Kalendervierteljahres (Quartals) abzugeben.  
Die Pflicht zur Abgabe der Erklärung beruht auf [§ 7 Absatz 1 der Satzung der Hansestadt Lüneburg über die Erhebung einer Steuer auf Übernachtungen in Beherbergungsbetrieben (Beherbergungssteuer) vom 8. November 2023.](LINK)

### Welche Unterlagen werden benötigt?

Falls Sie unseren [Online-Dienst zur Beherbergungssteuer](LINK) nicht nutzen können, müssen Sie die Erklärung per Vordruck vornehmen.  
Sie können den Erklärungsvordruck zur Beherbergungssteuer per E-Mail unter [beherbergungssteuer@stadt.lueneburg.de](LINK) anfordern.
