# Juleica (Jugendleiter-Card)

### Allgemeine Informationen
Die [Juleica](LINK) (Jugendleiter-Card) ist ein Aushängeschild der niedersächsischen Jugendarbeit. Sie ist der bundesweit einheitliche Ausweis für ehrenamtliche Mitarbeit in der Jugendarbeit. Sie bürgt für Qualität, weil sie nur im Zusammenhang mit einer bestandenen Prüfung ausgehändigt wird. Für ihre Inhaberin oder ihren Inhaber bringt sie Vorteile, ebenso für Vereine und/oder Organisationen, in denen sich diese Personen engagieren.
